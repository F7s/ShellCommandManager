# 国际化回归清单

每次修改 UI 或文案后，建议按下面清单回归：

1. 语言切换
- 在命令栏切换 `简体中文` / `English` 后，标题栏和主要按钮文案立即变化。
- 关闭应用再打开，语言保持上次选择。

2. 主界面静态文案
- 标题、命令配置区、模板区、已保存命令区文案均与当前语言一致。
- 输入框 Placeholder 与按钮文本与当前语言一致。

3. 运行与校验提示
- 名称为空、命令为空、路径不存在、数字参数非法等提示在两种语言下都正确。
- 运行成功/失败状态提示在两种语言下都正确。

4. 模板与弹窗
- 模板代码导入弹窗标题、按钮、占位提示在两种语言下正确。
- 运行前参数弹窗标题、按钮、文件/文件夹选择按钮文案正确。

5. 稳定性
- 切换语言后可正常保存命令、运行命令、导入模板。
- 应用启动不崩溃（重点检查启动阶段国际化逻辑）。
